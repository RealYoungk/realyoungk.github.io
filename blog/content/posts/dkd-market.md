## 시장 개요

당뇨병성 신장질환(Diabetic Kidney Disease, DKD)은 제2형 당뇨병의 주요 합병증으로, 전 세계 만성 신장질환(CKD)의 가장 흔한 원인이다.

| 지표 | 규모 |
|------|------|
| 글로벌 DKD 시장 (2024) | $2.8B ~ $5.6B |
| 글로벌 DKD 시장 전망 (2034) | $5.0B ~ $9.9B |
| CAGR | 약 5~6% |

> [!info] 미충족 의료 수요
> 기존 표준치료(ACEi/ARB + SGLT2i)로도 DKD 진행을 완전히 억제하지 못하며, 상당수 환자가 말기 신부전(ESKD)으로 진행한다. 새로운 기전의 치료제에 대한 수요가 크다.

---

## 표준 치료 체계

현재 DKD 치료는 단계적 병용요법으로 구성된다.

### 1차: ACE 억제제 / ARB (약 20년간 표준)

- RAAS(레닌-안지오텐신-알도스테론 시스템) 차단
- 거의 모든 DKD 환자에게 기본 처방
- 한계: 단독으로는 질환 진행 지연 불충분

### 2차: SGLT2 억제제 (최근 10년 추가)

- 대표 약물: 다파글리플로진, 엠파글리플로진
- 혈당 강하 + 신장 보호 이중 효과
- KDIGO/ADA 가이드라인 권고: ACEi/ARB에 추가 병용

### 3차: MRA — 케렌디아 (2021~ 신규 추가)

- 비스테로이드성 MR 길항제
- 염증/섬유화 억제
- **3제 병용요법의 세 번째 축**으로 자리매김 중

```
현재 표준치료 흐름:
ACEi/ARB (기본) → + SGLT2i (추가) → + MRA/케렌디아 (추가)
```

## 케렌디아 (Kerendia, 피네레논)

### 기본 정보

| 항목 | 내용 |
|------|------|
| 성분명 | 피네레논 (Finerenone) |
| 개발/판매사 | **바이엘 (Bayer)** |
| 분류 | 제3세대 비스테로이드성 선택적 MR 길항제 (ns-MRA) |
| FDA 승인 | 2021.07 — 제2형 당뇨병 동반 CKD |
| FDA 추가 승인 | 2025.07 — 좌심실 박출률 ≥40% 심부전 (HFpEF) |
| 한국 허가 | 2022.05 식약처 허가, 2024.02 급여 적용 |
| 투여 | 경구 1일 1회 (10mg / 20mg) |

### 작용 기전

무기질 코르티코이드 수용체(MR)에 선택적으로 결합하여 차단 → 염증, 섬유화 관련 유전자 발현 억제.

| 세대 | 약물 | 특징 |
|------|------|------|
| 1세대 | 스피로놀락톤 | 비선택적 스테로이드성 — 여성형 유방 등 부작용 |
| 2세대 | 에플레레논 | 선택적 스테로이드성 — 결합력 약함 |
| **3세대** | **피네레논 (케렌디아)** | **비스테로이드성 선택적** — 심장/신장 균등 분포 |

이전 세대 MRA(스피로놀락톤, 에플레레논)는 신장에 더 많이 축적되어 고칼륨혈증 위험이 높았으나, 피네레논은 심장과 신장에 균등하게 분포하여 상대적으로 위험이 낮다.

### 임상 근거

**핵심 임상시험:**

| 시험 | 대상 | 주요 결과 |
|------|------|-----------|
| FIDELIO-DKD | CKD + T2D (5,674명) | uACR -31~32%, 신장 복합 종점 18% 위험 감소 |
| FIGARO-DKD | CKD + T2D (7,437명) | 심혈관 복합 종점 13% 위험 감소 |
| FIDELITY (통합분석) | 13,000명+ | 신장·심혈관 이중 보호 확인 |
| FINEARTS-HF | HFpEF (6,001명) | 심부전 악화/사망 16% 위험 감소 |

### 매출 추이

| 시점 | 글로벌 매출 | 비고 |
|------|------------|------|
| 2023년 | ~€270M (~$300M) | +152% YoY |
| 2024년 | 빠른 성장 지속 | 분기별 +60~96% YoY |
| 피크 매출 목표 | **€3B (~$3.3B)** | 신장 + 심부전 이중 적응증 |

## 고칼륨혈증 — 케렌디아의 핵심 리스크

### 발생률

| 구분 | 케렌디아 | 위약 |
|------|---------|------|
| K⁺ > 5.5 mmol/L | **21.4%** | 9.2% |
| K⁺ > 6.0 mmol/L | **4.5%** | 1.4% |
| 치료 중단 | **1.7%** | 0.6% |
| 입원 | **0.9%** | 0.2% |

*FIDELIO-DKD, 2.6년 추적 기준. 고칼륨혈증 관련 사망은 0명.*

### 고칼륨혈증이 위험한 이유

고칼륨혈증(혈중 K⁺ > 5.0 mEq/L)은 **심장 전기 전도에 직접 영향**을 미친다.

**진행 순서:**
1. 뾰족한 T파 (peaked T wave)
2. PR 간격 연장, QRS 폭 증가
3. P파 소실
4. Sine wave 패턴 → **심정지**

- 대부분 **무증상**으로 진행하여 발견이 늦을 수 있음
- 중증 고칼륨혈증(> 6.5 mEq/L) 미치료 시 **사망률 최대 67%**
- 원내 심정지 위험 2배 증가

### 처방 제한

| 조건 | 제한 |
|------|------|
| K⁺ > 5.0 mEq/L | **투여 시작 금지** |
| eGFR < 25 | 투여 권장하지 않음 |
| 강력한 CYP3A4 억제제 병용 | 금기 |

투여 중 **4주마다 칼륨/eGFR 검사** 필수 — 의료진과 환자 모두에게 부담.

> [!warning] 실질적 영향
> DKD 환자 중 상당수가 이미 칼륨 수치가 높거나(ACEi/ARB 병용 효과), 신기능이 낮아 케렌디아를 사용할 수 없다. 이 환자군이 바로 CU01 같은 비-MRA 기전 약물의 타겟 시장이다.

## 경쟁 파이프라인

### 주요 경쟁약물

| 약물 | 기전 | 개발사 | 단계 | 특징 |
|------|------|--------|------|------|
| **케렌디아** | ns-MRA | 바이엘 | 시판 | 고칼륨혈증 리스크, 이중 적응증(DKD+HF) |
| **세마글루타이드** | GLP-1 RA | 노보 노디스크 | 승인 추진 중 | 심혈관 사망 24% 감소, DKD 적응증 확대 |
| **에삭세레논** | MRA | 다이이치 산쿄 | 일본 중심 | 케렌디아 유사 기전 |
| **Zibotentan+다파** | ETA 길항제+SGLT2i | 아스트라제네카 | 임상 3상 | 병용요법 전략 |
| **CU01** | Nrf2 활성제 | 큐라클 | 임상 2b상 완료 | 고칼륨혈증 위험 없음, 기전 차별화 |

### CU01의 시장 포지션

**케렌디아 vs CU01 핵심 차이:**

| 항목 | 케렌디아 | CU01 |
|------|---------|------|
| 기전 | MR 수용체 차단 (downstream) | Nrf2 활성화 — 산화 스트레스 제거 (upstream) |
| 고칼륨혈증 | 14% 발생 | MRA 아님 → 위험 없음 |
| 모니터링 | 4주마다 K⁺/eGFR 검사 필수 | 부담 낮음 (추정) |
| 처방 제한 | K⁺ > 5.0, eGFR < 25 금기 | 제한 적음 (추정) |
| uACR 감소 | -31~32% (4개월) | -21~22% (24주, 2b상) |
| 임상 단계 | **시판 중** | **2b상 완료** |
| 병용 가능성 | SGLT2i와 병용 | 케렌디아와도 병용 가능 (다른 경로) |

**CU01이 노릴 수 있는 시장:**
1. 케렌디아 금기 환자 (K⁺ > 5.0, eGFR < 25) — DKD 환자의 20~30% 추정
2. 모니터링 부담으로 MRA 기피하는 환자
3. 케렌디아와의 병용요법 (4제 병용: ACEi/ARB + SGLT2i + MRA + Nrf2)
4. 조기 CKD 단계 예방적 접근

> [!info] 참고: Bardoxolone 선례
> CU01과 유사한 Nrf2 활성제인 bardoxolone methyl(Reata/바이오젠)은 BEACON 시험(2013)에서 심부전 고위험군의 체액 과부하로 조기 종료된 이력이 있다. CU01의 3상 설계에서 이 리스크를 어떻게 관리하는지가 중요한 관건이다.
